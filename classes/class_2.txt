class DinningRoom

Attributes:
busy (Boolean)
number_of_tables (Integer)
shifts_available (Array)
server_greeting (String)

Methods:
is_busy (Checks to see if the dining room is busy; if busy is true, is_busy returns true; otherwise is_busy returns false)
remove_table (uses number_of_tables, takes away one table, and reassigns new value to number_of_tables)
add_shifts (uses shifts_available, adds open shifts, and reassigns the new value to shifts_available)
prints_server_greeting (uses server_greeting and says a friendly greeting to guest with exclamation)
